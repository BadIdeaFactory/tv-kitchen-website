[TV Kitchen](https://github.com/tvkitchen) is an open source framework that you can use to unlock the data inside of video.

Configure a countertop to turn videos into streams of new data, and then use that data to create applications, journalism, research, and art.

## Get Started

Read the [Quick Start](/docs/quick-start) guide if you want to set up a TV Kitchen to experiment yourself.

## Use a Recipe

We are working on more recipes, but for now check out the [Caption Extraction recipe](/docs/recipes/extracting-captions) to get some inspiration.

## Contribute

Visit the [Contributing Overview](contributing/overview) to learn more about how you can help improve the tools and share your creations.
